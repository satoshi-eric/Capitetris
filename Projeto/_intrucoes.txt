Para executar o jogo, é necessário instalar o xampp na máquina. 
Após a instalação, iniciá-lo e ativar o apache server e o mysql
Agora, é necessário criar o banco de dados. Para isso, acessar o phpmyadmin no localhost
http://localhost/phpmyadmin/index.php
Agora, criar o banco de dados de nome capitetris. Para isso, no painel esquerdo do phpmyadmin em Nome da base de dados, 
inserir capitetris e clicar em criar
Com isso, a base de dados será criada. Agora, é necessário executar um script para criação das tabelas.
Ele está em ./database/capitetris.sql da pasta de projeto
Clicar em importar no phpmyadmin e escolher arquivo. Escolher o arquivo capitetris.sql e clicar em executar.
Com isso, as tabelas serão criadas.
Agora, o sistema estará disponível para uso. Criar uma conta e logar.
